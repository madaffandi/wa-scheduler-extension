Privacy Policy

Last Updated: June 2026

WhatsApp Message Scheduler is designed to help users schedule messages for WhatsApp Web.

Information We Collect

The extension may store the following information locally in the user's browser:

  • Scheduled message content
  • Scheduled delivery date and time
  • Extension preferences and settings

The extension does not collect, transmit, sell, rent, or share user data with the developer or any third party.

Data Storage

All scheduled messages and settings are stored locally using Chrome Extension storage and remain under the user's control.

Third-Party Services

This extension interacts with WhatsApp Web solely to deliver messages scheduled by the user.

The extension does not use analytics, advertising, tracking technologies, or external servers.

Data Sharing

No user data is sold, transferred, or disclosed to third parties.

Security

The extension operates entirely within the user's browser and does not maintain any remote database containing user information.
